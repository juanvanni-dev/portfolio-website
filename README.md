# juanjosevanni.vercel.app 🌐

Personal portfolio website for **Juan José Vanni** — Web Developer & Designer based in Santa Fe, Argentina. Bilingual (EN/ES), dark mode, scroll animations and a modern editorial design.

**[→ Live Site](https://juanjosevanni.vercel.app/)**

---

## About This Project

This is my personal portfolio — designed and built from scratch to reflect both my technical skills and my approach to web projects: clean code, intentional design and real business value.

The site is bilingual (English / Spanish), fully responsive and built without any frameworks — just semantic HTML, modern CSS and vanilla JavaScript.

---

## Features

- 🌍 **Bilingual** — full EN/ES language toggle
- 🌙 **Dark / Light mode** — user preference toggle with smooth transition
- ✨ **Scroll animations** — sections animate on entry for a polished feel
- 📱 **Mobile-first** — optimized for all screen sizes
- ⚡ **Performance-focused** — no unnecessary dependencies
- 🔍 **SEO basics** — semantic HTML, meta tags and descriptive structure
